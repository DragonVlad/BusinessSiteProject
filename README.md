# Business Site 

Проект представляет собой статичную верстку веб-сайта-визитки для вымышленной компании. Он содержит **HTML и CSS** верстку одностраничного бизнес-сайта. Основная цель проекта — демонстрация навыков верстки, создания современного дизайна. 

Макет включает в себя стандартные разделы:
- **Хедер** с контактами и логотипом;
- **Главный экран** с описанием услуг;
- **Раздел "Проекты"**;
- **Футер** с контактной информацией.
